# Flutter JSONPlaceholder example

Flutter app example of using fake REST API.

[JSONPlaceholder](https://jsonplaceholder.typicode.com/): 
- users
- posts
- comments
- albums
- photos
- todos

Saving data to cache is implemented using SharedPreferences
